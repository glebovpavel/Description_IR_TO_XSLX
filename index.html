Interactive Report to Excel v2

ORACLE APEX plugin

Export your Interactive Reports to Excel

The "GPV Interactive Report to MSExcel" APEX plugin lets you get results of your Interactive Report in Excel-format, with correct displaying of Strings, Number and Dates.

Supported Interactive Reports features

  • (new in v2.0) APEX 5 support
  • (new in v2.0) Different Rendering Engines
  • Filtering and Sorting
  • Control Breaks
  • Computations
  • Aggregations
  • Highlighting
  • Auto Column Widths
  • Date/Number Formats
  • Custom File Name
  • Can be easy used for all IR
  • (new in 2.20) Great performance
